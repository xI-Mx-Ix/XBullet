# XBullet - Physics Engine for Minecraft 1.20.1 Forge

**XBullet** is an experimental **Physics Engine** for **Minecraft 1.20.1** built with **JBullet**. The mod is still in **Beta** and aims to introduce more **realistic physics** to Minecraft, including features like realistic **collisions** and **physics-based movement**.

**Warning:** This mod is still in development and may contain many **bugs**. It is intended for **testing purposes** only. Please report any **bugs, crashes, or unexpected behavior** to help improve the mod.

## Features

- Implements a **realistic physics engine** in Minecraft.
- Built using **JBullet**, a Java-based physics library.
- Realistic **collisions** and **movement** for blocks and entities.
- Supports **Minecraft Forge 1.20.1** version.

## Usage

With the `/summon xbullet:block_collision` command, you can spawn a stone block that behaves according to the physics engine, including realistic **collisions** and **movement**.
